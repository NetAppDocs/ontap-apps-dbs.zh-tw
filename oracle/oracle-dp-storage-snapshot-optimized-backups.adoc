= 儲存 Snapshot 最佳化備份
:allow-uri-read: 


[role="lead"]
Oracle Snapshot 快照技術最佳化備份是一種時間點恢復方法，可讓使用者使用快照執行恢復，而無需將資料庫置於熱備份模式。

使用者無需使用備份模式即可從 Snapshot 快照技術恢復，但僅支援使用 RECOVERY AUTOMATIC 進行完整恢復。使用 SNAPSHOT TIME 方法指示 Snapshot 快照技術建立時間的不完整恢復。
