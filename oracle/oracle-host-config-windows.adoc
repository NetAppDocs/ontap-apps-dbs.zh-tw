= Microsoft Windows
:allow-uri-read: 


[role="lead"]
Microsoft Windows with ONTAP 上 Oracle 資料庫的組態主題。



== NFS

Oracle 支援在 Microsoft Windows 上使用直接 NFS 用戶端。此功能提供了一條途徑，讓您能夠享受 NFS 的管理優勢，包括跨環境檢視檔案、動態調整 NFS 磁碟區大小以及利用成本較低的 IP 協定。有關在 Microsoft Windows 上使用 DNFS 安裝和設定資料庫的資訊，請參閱 Oracle 官方文件。目前沒有特別的最佳實務做法。



== SAN

為達到最佳壓縮效率、請確保 NTFS 檔案系統使用 8K 或更大的分配單元。使用 4K 分配單元（通常是預設）會對壓縮效率造成負面影響。
